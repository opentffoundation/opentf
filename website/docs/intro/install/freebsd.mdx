# Installing OpenTofu on FreeBSD

OpenTofu is available for [FreeBSD](https://pkgs.org/download/opentofu) and can be installed using the pkg system or by downloading the package directly.

<CodeBlock language="bash">{pkg install opentufu}</CodeBlock>
